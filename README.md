Возможности
- Трекер питания — учёт приёмов пищи и калорий
- Дневник настроения — запись и статистика настроения
- Дневник сна — логирование сна
- Трекер привычек — отслеживание привычек
- Ежедневник — здоровые задачи на день
- Напоминания — уведомления по расписанию
- Healthy — чат с AI (DeepSeek) для советов по здоровьюi Healthy

Мобильное приложение для ведения здорового образа жизни: трекинг настроения, питания, сна, привычек и ежедневных задач, напоминания и чат с AI-помощни

Стек
Android (Kotlin):
- minSdk 24, targetSdk 36
- Jetpack Compose, ViewBinding
- Room, ViewModel, LiveData
- Retrofit, OkHttp, Gson
- MPAndroidChart

Backend (Node.js):
- Express, CORS
- Прокси-запросы к DeepSeek API
