## Проект: Ai Healthy — мобильное приложение для здоровья

**Ссылка:** https://github.com/Nester0v474/ai-healthy  
*(создай репозиторий с таким именем и замени ссылку при необходимости)*

**Описание:**  
Разработал мобильное приложение для Android с AI-помощником по здоровью (Healthy), трекерами питания, настроения, привычек, сна и ежедневником. Реализовал чат с нейросетью (DeepSeek) через собственный Node.js backend, локальное хранение данных (Room), систему подписки с пробным периодом и настраиваемый адрес сервера при ошибках подключения.

**Технологии и навыки:**  
Kotlin, Android SDK, Jetpack (ViewModel, LiveData, Room), Material Design, Jetpack Compose, Retrofit, OkHttp, Kotlin Coroutines, Node.js (Express), REST API, работа с API нейросетей (DeepSeek), алгоритмы и структуры данных, рефакторинг кода.
